# Calculadora de IMC

## Descrição

Este projeto foi desenvolvido utilizando ASP.NET Core MVC.

A aplicação recebe o peso e a altura do usuário por meio de um formulário e realiza o cálculo do Índice de Massa Corporal (IMC) no Controller, utilizando processamento Server Side.

Após o cálculo, o sistema exibe o valor do IMC e sua classificação correspondente.

## Funcionalidades

- Receber peso do usuário
- Receber altura do usuário
- Calcular IMC no Controller
- Exibir resultado do IMC
- Exibir classificação:
  - Abaixo do peso
  - Peso normal
  - Sobrepeso
  - Obesidade

## Tecnologias Utilizadas

- ASP.NET Core MVC
- C#
- Razor Views
